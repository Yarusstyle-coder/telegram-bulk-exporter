tdl session

Telethon (для списка чатов) и tdl (для скачивания медиа) хранят сессии раздельно. После Telegram-логина нужно ещё один раз авторизовать tdl. Это разовое действие — сессия сохранится в %USERPROFILE%\.tdl\data\default.

К Telegram dashboard

Статус

QR / SMS: tdl на Windows нужен реальный TTY, поэтому при «Запустить» откроется отдельное окно cmd — отсканируйте QR / введите код там, окно закроется автоматически.

Desktop: tdl сам найдёт tdata установленного Telegram Desktop (моментально, без QR).

Если окно не открылось (например запускаете под IDE без права на новые консоли) — выполните в PowerShell прямо сейчас: 

.\tools\tdl\tdl.exe -n default login -T qr

после «Login successfully!» вернитесь на эту страницу — статус автоматически переключится в «success».

Нажмите «Запустить». Для режима QR / SMS откроется отдельное окно tdl где будет интерактивный шаг (на Windows иначе tdl не умеет отдавать QR в pipe).
